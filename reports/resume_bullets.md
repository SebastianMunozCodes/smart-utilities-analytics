# Resume Bullet Drafts

## Project Title

Smart Utilities Analytics | Python, Pandas, SQLite, SQL, Matplotlib

## Resume Bullets

- Built an end-to-end electricity analytics project using Python, Pandas, SQLite, SQL, and Matplotlib to clean, analyze, visualize, and report on monthly EIA electricity sales and revenue data.

- Developed a reproducible data workflow that converted raw electricity data into a cleaned CSV, loaded the cleaned dataset into a SQLite database, reproduced Pandas summaries with SQL queries, and generated charts and a written Markdown report.

- Analyzed state-level electricity sales, revenue, and price trends, identifying Texas as the highest total sales state, California as the highest revenue state, and Hawaii as the highest average price state.

## Short Project Description

Smart Utilities Analytics is a data analytics project that uses public EIA electricity data to demonstrate data cleaning, Pandas analysis, SQL querying, SQLite database loading, Matplotlib visualization, and report generation.

## Application Story

After seeing that the role involved data management and analytics, I built a utilities-focused analytics project to strengthen my Python, SQL, and data cleaning skills. The project uses public EIA electricity data, cleans it with Pandas, stores it in SQLite, and compares Pandas and SQL summaries for sales, revenue, pricing, and monthly trends. This helped me connect my data management coursework to a real-world utilities-style analysis workflow.
